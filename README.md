# Futebol Semanal

# ⚽ Sistema de Gestão para Grupo de Futebol

Sistema desenvolvido para auxiliar na organização e gestão de um grupo de futebol, oferecendo controle completo de jogadores, estatísticas, finanças e relatórios.

## 📌 Funcionalidades

- 📊 Controle de estatísticas dos jogadores
- 📈 Geração de relatórios
- 💰 Controle de caixa (entradas e saídas)
- 👥 Gerenciamento de participantes
- 📅 Organização de jogos e atividades
